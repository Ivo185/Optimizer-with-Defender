# 🛡️ Optimizer with Defender — Windows почистващ инструмент

Мощен PowerShell инструмент за оптимизация на Windows системи с интегрирано антивирусно сканиране чрез Microsoft Defender. Поддържа **два езика** (Български и English) и предлага напълно персонализируемо почистване.

---

## 📋 Описание

**Optimizer with Defender** автоматизира почистването на временни файлове, кошчето и системния кеш, като едновременно може да стартира пълно антивирусно сканиране. Преди всяко почистване се създава **точка за възстановяване** за максимална сигурност.

### Какво прави скриптът:

- Автоматично иска **администраторски права** при стартиране
- Поддържа **Български и English** интерфейс
- Създава **точка за възстановяване** преди почистване
- Стартира **пълно сканиране с Microsoft Defender** (по избор)
- Почиства:
  - 🗂️ Потребителски `%TEMP%` файлове
  - 🗂️ Системен `C:\Windows\Temp`
  - 🗂️ `Prefetch` кеш
  - 🗑️ Кошче (Recycle Bin)
  - ⚙️ `WinSxS` компоненти чрез DISM (по избор)
- **Smart Scan** — открива и предлага изтриване на остатъчни папки от деинсталирани приложения
- Генерира **лог файл** с два нива на детайлност (по избор)
- Показва **освободеното дисково пространство** в MB
- Рестартира `explorer.exe` накрая (по избор)
- Издава звуков сигнал при завършване

---

## ⚙️ Изисквания

- Windows 10 / 11 (препоръчително)
- PowerShell 5.0 или по-нова версия
- **Задължително: Администраторски права** (скриптът ги иска автоматично)
- Microsoft Defender (активен, за функцията за сканиране)

## 🚀 Как да използваш

### Метод 1 — PowerShell директно

```powershell
.\Optimizer_with_Defender.ps1
```

> Скриптът автоматично се рестартира с администраторски права, ако е нужно.

### Метод 2 — Десен клик

1. Кликни с десен бутон върху файла
2. Избери **"Run with PowerShell"**

### Метод 3 — Ако скриптовете са блокирани

```powershell
powershell -ExecutionPolicy Bypass -File .\Optimizer_with_Defender.ps1
```

## 📊 Функции по избор

| Функция | Описание |
|---------|----------|
| 📄 Лог файл | Записва всички действия в `Optimization_Report.txt` |
| 🛡️ Defender Scan | Пълно антивирусно сканиране преди почистване |
| 🔄 Рестарт Explorer | Освежава Windows UI след почистване |
| ⚙️ WinSxS | Намалява размера на системните компоненти (бавно) |
| 🔍 Smart Scan | Интерактивно намиране на остатъци от деинсталирани програми |

---

## ⚠️ Важни бележки

- **Smart Scan** показва папки, които **не съответстват** на инсталирани програми — винаги проверявай преди изтриване
- **WinSxS** оптимизацията може да отнеме **10–30 минути**
- Defender Scan се пропуска автоматично, ако вече има активно сканиране
- Лог файлът се записва в **същата папка като скрипта**
